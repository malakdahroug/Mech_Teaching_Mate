Simulated PLC

You can use this tool to simulate automation sequences.

The tool will take any sequence (in according to the guidance available on the Instructions page) and simulate it using one of the available PLCs. The list of available PLCs contains all PLCs that are currently not executing sequences.

The PLC will execute the sequence either in FluidSIM or any other system it is connected to. To simulate the PLC, please provide an automation sequence you want to simulate, choose one of the available PLCs from the available PLCs list and press the START EXECUTION button.

If no PLCs ara available, they can be stopped by selecting the PLC from the list of unavailable PLCs and pressing the STOP EXECUTION button. Before doing that, make sure no one else is currently trying to observe the sequence simulation.
