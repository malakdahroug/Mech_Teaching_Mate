Simulated PLC

You can use this tool to simulate automation sequences.

The tool will take any sequence (in accordance with the guidance available on the Instructions page) and simulate it using one of the available PLCs. The list of available PLCs contains all the PLCs that are currently not executing sequences.

The PLC will execute the sequence either in FluidSIM or any other virtual or physical system it is connected to. To simulate the PLC, please provide an automation sequence you want to simulate, choose one of the available PLCs from the available PLCs list and press the START EXECUTION button.

If no PLCs ara available, they can be stopped by selecting the PLC from the list of unavailable PLCs and pressing the STOP EXECUTION button. Before doing that, make sure no one else is currently trying to observe a sequence simulation using the selected PLC.

Currently, the simulated PLC is only capable of executing sequences with all actuators starting in their retracted positions. Additionally, the simulated PLC only supports a maximum of one level of nested repeating sequences - only for sequences that start and end with square brackets. Nested sequences that are supported must be repeating infinitely (without a number of repetitions specified at the end).

Correct sequences:
  • [A+,[B+,B-]^3,[A-,A+]^3,A-]
  • [A+,[B+,B-]^3,A-]
Incorrect sequences:
  • [A+,[B+,B-,[A-,A+]^2]^3,[A-,A+]^3,A-]
  • A+,A-,[A+,[B+,B-]^3,A-]
