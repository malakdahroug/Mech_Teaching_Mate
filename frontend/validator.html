Sequence validator

You can use this tool to validate automation sequences to be used in the system

The tool will take any sequence (in accordance with the guidance available on the Instructions page) and return information about validity of the sequence. If an incorrect sequence is provided, a list of errors will be returned instead. Each action containing an error will be listed in a separate line.

To validate the sequence, please input the sequence to validate in the Sequence field and press the VALIDATE SEQUENCE button.
