Code generator

You can use this tool to generate SCL code for the given PLC sequence. The generator can also generate code with mistakes that can be solved.

The tool will take any sequence (in according to the guidance available on the Instructions page) and generate the correct solution, incorrect solution and the list of PLC tags (for sequences that are inputted in the sequence field) that can be imported in TIA Portal. You can define your own PLC tags by creating a sequence in the Project tab and assigning corresponding components to it.

To generate the sequence, please provide it in the Sequence field, select the complexity of errors (either None (for no errors to be generated), Easy, Medium or Hard) and press the GENERATE CODE button. If you selected a complexity of Errors to generate, the tool will display the incorrect code. You will be able to see the correct code by pressing the SHOW SOLUTION button.

All of the code snippets generated (correct, incorrect and XML Tags) can be copied to clipboard by pressing the corresponding Copy to clipboard button.

Currently, the generator is only capable of generating code for sequences with all actuators starting in their retracted positions.
