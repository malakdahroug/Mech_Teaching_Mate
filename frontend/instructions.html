Instructions

This page contain all instructions on how to prepare the sequence for the generation. The sequence has to be satisfying set of rules otherwise the Generator, Validator or Simulated PLC might return errors.

Sequence description

  • Sequence can contain both lowercase and uppercase letters
  • Sequence can contain actuators marked with A-Z letters
  • Sequence can contain timers written in the following form
    • XS - where X is an integer or floating point number
    • TS
    • T+XS - where X is an integer or floating point number
  • Sequence can contain repetitive parts indicated with square brackets e.g. [ SEQUENCE TO REPEAT ]^x - where X is an integer number greater than or equal to 2
  • Sequence can contain concurrent parts indicated with round brackets e.g. (concurrent actuations)
  • Sequence can only start with all cylinders retracted
Example of valid sequences

The sequences listed below will get correctly verified by the Mechatronics Teaching Mate. Code and errors currently can be generated only for the simple sequences - the ones that do not include timers, repetitive and concurrent parts.

  • A+,A-,B+,B-
  • C+,B+,TS,A+,B-,C-,A-
  • A+,B+,(A-,B-),(C+,B+),C-
  • A+,(B+,2S),A-,B-
  • [A+,(B+,2S),A-,B-]
  • A+,B+,[A-,B-,C+,B+,C-,A+]^2
  • A+,B+,[A-,B-,C+,B+,C-,A+]^n+2
  • A+,B+,[(A-,B-),C+,(B+,C-),A+]^20
  • [A+,B+,C+,D+,[(A-,B-),A+,(A-,B+),A+]^3,D-,B-]
  • [A+,B+,C+,D+,A-,B-,C-,D-,[(A+,C+),A-,(C-,B+),B-]^3,B+,D+,D-,B-]

Examples of invalid sequences

The sequences listed below won't be successfully validated using Mechatronics Teaching Mate validator. Also, the code for the given sequences won't get generated.

  • A+A-B+B-
  • C+,B+,s,A+,B-,C-,A-
  • A+,B-,A-,B+),(C+,B-),C-
  • A+(B+,S),A-,B-
  • [A+,(B+,2SS),A-,B-]
  • A+,B+,[A-,B+,C+,B-,C-]^0.4
  • A-,B+,[A-,B-,C+,B+,C-,A+]^n+2
  • A+,B-,[(A-,B+),C+,(B-,C-)]20
  • [A+,B+,C+,D+,[(A-,B-),A+,(A-,B+]^3,D-,B-]
  • [A+,B+,C+,D+,A-,B-,C-,D-,[(A+,C+),A-,(C-,B+)^3,D-,B-]
